# SSP
SSP - Simple System Posts, it is a little project with the technologies SEVN(Sequelize, Express, Vuejs, Nodejs), like first travel!

 - **Dict root**
 
    - `npm install`

- **Dict front_end**

    - `npm install`

- **Dict root run serve**

    - `npm start`

- **Dict front_end run serve**

    - `npm run serve`
